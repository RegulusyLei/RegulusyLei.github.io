webpack的使用(二)

一月 15, 2017 Study

加载器简介

webpack准备的一些预处理工具

加载器

1. babel-loader
    ++ 用来编译jsx和ES6到原生js
        -- jsx语法: 允许html和js混写
    ++ 安装
        npm install babel-loader --save-dev
        npm install babel-core babel-preset-es2015 babel-preset-react --save-dev
    ++ 修改配置
        -- 与入口和出口同级
        -- module: {
            loaders: [
                //用来处理jsx和es6语法
                {
                    test: /\.jsx?$/,
                    loader: 'babel',
                    query: {
                        presets: ['es2015', 'react']
                    }
                }
            ]
        }
    ++ 启动运行
2. css-loader
    ++ 运行在js中引用css文件
    ++ 安装
        npm install css-loader style-loader --save-dev
    ++ 修改配置
        -- 与上边一样,继续添加进loaders数组
        //处理css引用
        {
            test: /\.css$/,
            loader: 'style!css' //注意,多个加载器用感叹号连接,执行顺序是从右往左
        }
    ++ 启动运行
